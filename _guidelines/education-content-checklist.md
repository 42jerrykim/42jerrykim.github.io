# 교육 콘텐츠 크로스체크 가이드라인 (이동됨)

이 문서의 내용은 [`educational-content-writing` 스킬](../.claude/skills/educational-content-writing/SKILL.md) **§7 "내용 완결성 — 석학(SME) 기준"** 으로 편입되었다 (2026-07-02). 글 작성·보강 시 스킬이 자동 적용되므로 이 파일은 더 이상 갱신하지 않는다.

편입 시 다른 스킬과의 중복은 다음과 같이 정리되었다:

- **7대 완결성 축(C1~C7)** + 작성 전 준비 절차 → 스킬 §7.1 (역사·인용·수치 검증은 기존 §4.1/§4.3/§4.4를 참조로 위임)
- **과목별 필수 커리큘럼** → 스킬 §7.2 (어휘는 `vocabulary-post-writing`, 알고리즘은 `algorithm-post-writing` 위에 얹는 추가 항목만 유지; 수학/과학/역사는 전담 스킬이 없어 전량 유지)
- **태그=본문 일치 규칙** → 스킬 §7.3
- **작성 전/후 체크리스트** → 스킬 §8의 "내용 완결성" 블록으로 병합
- **반복 결함 패턴** 중 기존 스킬이 이미 다루는 것(필러 금지=§2.5, 미검증 수치=§4.4)은 중복 제거
